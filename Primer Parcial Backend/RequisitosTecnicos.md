***Requisitos Técnicos del proyecto:***
El desarrollo debe cumplir con:
● Uso exclusivo de las siguientes tecnologias:
    ○ Java 17
    ○ Java EE
    ○ Maven
    ○ Quarkus
    ○ PostgreSQL
    ○ JPA para persistencia
    ○ EJB3 para lógica de negocio
    ○ JAX-RS para exposición de servicios REST
● Arquitectura en capas:
    ○ Entidades
    ○ Servicios (EJB)
    ○ API REST
● Manejo adecuado de:
    ○ Transacciones
    ○ Validaciones
    ○ Excepciones
***API REST***
Se espera que el sistema exponga endpoints claros y bien estructurados para:
    ● Gestión de camiones
    ● Gestión de reservas
    ● Búsqueda de disponibilidad
El diseño de la API (rutas, métodos HTTP, payloads) queda a criterio del alumno.
